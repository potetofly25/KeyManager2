# KeyManager2 - アカウントマネージャー

ログインID,パスワードを管理するデスクトップツール  
パスワード生成機能あり  
保存するデータはSQLiteでローカルに保存  


# Requirement

* VisualStudio C#
* .NET9.0
* CommunityToolkit.Mvvm
* Avalonia UI MVVM App  https://avaloniaui.net/  


# Nuget

使用しているNugetパッケージ

* EntityFrameworkCore 9.0.11  https://learn.microsoft.com/ja-jp/ef/core/
